A) Server Requirements

Next Pos is web based application ,it requires following (below ) extensions should be installed to run properly .

Apache server running PHP 7.2 or 7.2*.* .

Extension list
  • PHP 7.2*.*
  • Mcrypt PHP extension
  • Mysqli PHP extension
  • MBString PHP extension
  • GD PHP extension
  • CURL PHP extension
  • OPEN SSL PHP extension
  • PDO PHP extension
  • TOKENIZER PHP extension
  • XML PHP extension
  • Ctype PHP extension
  • JSON PHP extension

Make Sure the listed extensions are installed on your hosting sever.

If your server has already installed these extensions then follow next step.
